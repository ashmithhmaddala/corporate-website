Features

Everything you need to
manage secrets at scale.

From small teams to enterprise organizations, Vault gives you the tools to secure, share, and sync your secrets across every environment.

Environment Management

Organize secrets by environment—development, staging, production. Each environment is isolated with its own access controls, so your production database credentials never end up in a developer's local setup.

  • Unlimited environments per project
  • Environment-level access controls
  • Clone environments with one click
  • Environment variables inheritance

Secret Versioning

Every change is tracked. Roll back to any previous version instantly. Know exactly when a secret changed and who changed it.

End-to-End Encryption

Secrets are encrypted at rest with AES-256 and in transit with TLS 1.3. Your data is never accessible to us, even if you wanted it to be.

Team Access Controls

Granular permissions at every level. Give contractors read-only access to staging. Give seniors full access to production. Revoke instantly.

CLI & SDKs

Our CLI injects secrets at runtime. SDKs for Node.js, Python, Go, Ruby, and more. Works with your existing workflow, not against it.

Audit Logs

Complete visibility into every action. Who accessed what, when, and from where. Export logs for compliance or pipe them to your SIEM.

Developer Experience

Built for the way
developers actually work.

No more context switching. No more copy-pasting from shared docs. Vault integrates into your terminal, your IDE, and your CI/CD pipeline.

Install the CLI 
$ npm install -g @vault/cli
Run with secrets 
$ vault run -- npm start
→ Injecting 12 secrets from production...
→ Server started on port 3000
Or use the SDK 
import { Vault } from '@vault/sdk';

const vault = new Vault();
const dbUrl = await vault.get('DATABASE_URL');
