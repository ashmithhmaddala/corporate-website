What's new in Vault

Product updates, new features, and improvements. Subscribe to get notified when we ship something new.

January 30, 2026

Native Kubernetes Secrets Sync

Sync your Vault secrets directly to Kubernetes clusters. Our new operator watches for changes and updates K8s secrets automatically—no more manual kubectl apply.

Supports namespaced secrets, configmaps, and works with Helm charts out of the box.

New
January 22, 2026

Improved Audit Log Filtering

Filter audit logs by user, action type, environment, and date range. Export filtered results to CSV for compliance reporting. Added full-text search across log entries.

Improved
January 15, 2026

CLI v2.4.0

New vault diff command to compare secrets between environments. Improved vault run performance with parallel secret fetching. Fixed an issue where the CLI would hang on Windows when using Git Bash.

New Fixed
January 8, 2026

Secret Comments

Add comments to individual secrets to document what they're for, where they came from, or when they should be rotated. Comments are searchable and visible to all team members with access.

New
December 18, 2025

Railway Integration

Sync secrets to Railway services and environments. Changes push automatically when you update secrets in Vault. Supports all Railway service types including databases and cron jobs.

New
December 10, 2025

Personal Access Tokens

Generate scoped personal access tokens for CI/CD and automation. Tokens can be limited to specific projects, environments, and actions. Set expiration dates for additional security.

New
December 2, 2025

Bulk Import/Export

Import secrets from .env files, JSON, or CSV. Export in the same formats. Makes migration from other tools (or no tools) much easier.

New
November 20, 2025

Dashboard Redesign

Fresh new look for the dashboard with improved navigation, better secret organization, and faster load times. Dark mode is now the default (finally).

Improved
November 12, 2025

Secret References

Reference one secret from another using ${SECRET_NAME} syntax. Great for building connection strings that combine host, port, and credentials.

New
November 5, 2025

Okta SSO Support

Added Okta as a SAML identity provider option. Team plan and above. Also fixed an issue with Azure AD group syncing.
